# Risk Signal Mining

This project identifies financial risk signals from credit card data
using rule-based analysis in Python.
Customers are grouped into risk segments based on their behaviour.

---

## Objectives
- Detect early risk signals
- Create a risk score
- Segment customers by risk level

---

## Dataset
- `credit_card_default.csv`

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Insights
- High-risk customers have higher default rates
- Credit utilization is a strong risk indicator
- Repayment behaviour helps identify financial risk
