# Customer Behaviour Analysis

This project analyzes customer behaviour using Python.
A simple rule-based **Customer DNA** approach is used to understand
customer value, loyalty, and purchasing patterns.

---

## Objectives
- Analyze customer demographics and behaviour
- Create customer value segments
- Generate business insights using data

---

## Dataset
- `customer_data.csv`

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Insights
- High-value customers spend more and show higher loyalty
- Promotions influence customer spending
- Customer value varies across segments
